# VSM Question Mapping

## Insight

The five VSM levels map cleanly to interrogative questions:

| S-Level | Element | Question | Domain |
|---------|---------|----------|--------|
| S5 | Water | WHO | Identity, purpose, soul |
| S4 | Fire | WHY | Intelligence, adaptation, strategy |
| S3 | Earth | HOW | Control, procedures, resources |
| S2 | Metal | WHEN | Coordination, timing, ordering |
| S1 | Wood | WHERE | Operations, execution, locus |

## Key Distinction

The "WHAT" is the whole agent — it spans all five levels. The agent IS the integrated system.

## Diagnostic Value

This framing sharpens diagnostics: when something's off, ask "which question isn't being answered?" rather than just "which element is imbalanced?"

- S5 excess → identity crisis (who are we?)
- S4 excess → constant pivoting (why are we doing this?)
- S3 excess → micromanagement (how do we control this?)
- S2 excess → bureaucracy (when do things get approved?)
- S1 excess → chaos (where is the work happening?)
