# Launch Fast vs CreatorOS — 3-Surface Gap

**Context:** Re-analyzed Launch Fast after building CreatorOS demo (5 modules, 31 tests, 52 assertions, 0 failures).

**Key gaps:** CreatorOS has the engine but missing the distribution surfaces. Launch Fast's three-surface pattern (Chrome extension + web dashboard + MCP server) is what turns an engine into a product people pay for.

**P0:** Chrome extension that overlays product intelligence on TikTok/Amazon pages. OV5's Clojure→CLJS pipeline handles this natively.

**P1:** MCP server exposing matching/scoring/profit/sources as AI-accessible tools. Web dashboard built as Reagent SPA.
