When extracting from lists with plist-get/cdr, always wrap with `(when (listp r) ...)` to prevent errors on non-list values. The pattern `(plist-get (or (cdr r) (plist-get r :scores)) :key)` can fail if `r` is not a list — guard with `when (listp r)` before the plist-get call. Prefer `(plist-get r :scores)` over `(cdr r)` as primary extraction since results are often plists, and use `(and (consp r) (cdr r))` as fallback only when confirmed cons.
