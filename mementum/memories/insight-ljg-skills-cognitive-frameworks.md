# ljg-skills cognitive frameworks beat OV5 surface scanning

Studied ljg-skills (https://github.com/lijigang/ljg-skills, 5.9k stars). Key insight: ljg provides **cognitive framework skills** (structured thinking methodologies) while OV5 provides **technical capability skills** (code generation, validation). ljg transforms how AI *thinks*; OV5 transforms what AI *does*.

## Five highest-leverage cognitive frameworks

1. **ljg-learn** — 8-dimension concept anatomy (history, dialectics, phenomenology, linguistics, formalization, existentialism, aesthetics, meta-reflection) → compress to epiphany
2. **ljg-paper** — 7-beat narrative spine (protagonist/dilemma/old-path/turning-point/solution/ending/core) + speed-read card + PhD advisor review
3. **ljg-paper-river** — Recursive citation tracing (5 layers back + forward) → problem evolution story
4. **ljg-think** — Vertical deep-drill to irreducible essence (mechanism → principle → axiom)
5. **ljg-roundtable** — Multi-persona structured debate with truth-seeking moderator + ASCII framework diagrams

## Key design patterns worth adopting

- **Trigger words** — Natural language invocation ("解剖概念", "读论文") > ontology routing
- **Visual output** — PNG cards, ASCII diagrams > text-only
- **Storytelling** — Papers as narratives > findings as extractions
- **Workflow chains** — paper → cards, word → infographic > isolated skills
- **Denote integration** — org-mode files with timestamps/tags > mementum memories

## Best integration path for OV5

1. **Adopt cognitive frameworks as research skills** — paper-storytelling, citation-tracing, concept-anatomy, deep-drill, multi-perspective
2. **Add trigger-word system** — Natural language skill invocation > keyword scoring
3. **Add visual output** — ASCII knowledge maps, org-mode reports
4. **Build interactive skill browser** — Emacs UI for skill graph visualization
5. **Add workflow composition** — Interactive tool to chain skills into workflows

## Strategic insight
ljg optimizes **human cognition** (thinking methodologies); OV5 optimizes **automated evolution** (skills improve through experiments). Integration: adopt ljg's cognitive frameworks as OV5 research skills → gives OV5 both depth (8-dimension analysis) and evolution (automated improvement).
